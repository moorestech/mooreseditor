# Project Overview

- **Purpose**: Tauri (Rust) + React + TypeScript desktop app for editing JSON master data based on dynamically loaded YAML schemas. Must never hard-code schema structures.
- **Domain**: Internal data tooling; focuses on generic schema-driven forms/tables with clipboard workflow.
- **Key Concepts**: Dynamic schema parsing, JSON editing views (form/table), clipboard operations, schema-based validation, auto-generated GUID fields.
